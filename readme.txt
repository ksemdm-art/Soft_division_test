Запуск:
1) устанавливаем node.js версии 14
2) устанавливаем зависимости, в командной строке: npm install
3) запуск сервера разработки, в командной строке: npm run dev

Разработка:
Должна вестись в папке ./src/client/pages/main, остальные желательно не трогать.

Задача:
Необходимо разработать SPA-приложение для ведения списка Сотрудников и отслеживания их дружественных связей.
Разрабатывается только клиентская часть, сохранять ничего в базу данных не нужно, все данные хранить на клиенте.

В приложении должно быть две страницы:
1) список Сотрудников
- список с полями: фамилия, имя, отчество, email и количество друзей
- должна быть возможность добавить нового Сотрудника
- должна быть возможность просмотреть информацию о существующем Сотруднике
- должна быть возможность отредактировать существующего Сотрудника
- должна быть возможность удалить существующего Сотрудника
2) данные о Сотруднике
- просмотр и редактирование полей: фамилия, имя, отчество, email
- возможность добавить в друзья ранее созданного Сотрудника
- возможность удалить из друзей ранее созданного Сотрудника
- возможность вызова функции, которая отобразит все циклические связи текущего Сотрудника
  пример:
    Вася добавил Петю, а Петя добавил Васю
    Вася добавил Диму, Дима добавил Тому, а Тома добавила Васю

Используемые технологии (все зависимости подключены и настроены):
Сервер:
- node.js
- express.js
Клиент:
- vue.js
- vue-router.js
- vuetify.js
